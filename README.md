# week2-code-challenge
SHOPPING LIST

this is a webpage that allows the user to :
input an item to add to the list,mark an item as purchased and clear the list
There are 3 buttons:Add to add items to the list
                    mark purchased to draw a line on the text


                    Clear list to remove everything from the list
CODE

The webpage has basic html structure,a css file and a javascript file.

JAVASCRIPT


The webpage uses javascript to add functionality to it.
It uses sn Array to store the data on the list container.
It also uses events and event listeners to add functionality to the buttons.
